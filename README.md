# Hr-workforce-analytics-dashboard

HR Workforce Analytics & Employee Attrition Dashboard
Overview

This project focuses on analyzing HR employee data to understand workforce distribution, employee attrition, salary trends, recruitment sources, satisfaction levels, and performance patterns. SQL was used for data analysis and KPI calculation, while Power BI was used to build an interactive business dashboard.

Business Problem

HR teams need to understand why employees leave, which departments have higher attrition, how salary varies across roles, and how employee satisfaction, performance, absence, and recruitment source impact workforce stability.

Results

The dashboard helps identify high-attrition areas, analyze workforce demographics, compare salary trends, and support HR teams in employee retention planning.
